## 電話番号表示の方針

### A. チャット内での案内（AI応答）

#### 緊急度：高（即座に案内）
**条件：**
- 身体的危険がある
- 犯罪の可能性がある
- 財産に重大な損害がある
- 精神的に追い詰められている
- ユーザーが「電話で相談したい」と明示

**表示内容：**
```
この状況は緊急性が高いため、お電話でのご相談をお勧めします。
専門スタッフが詳しくお話を伺い、適切な対応をご案内いたします。

📞 トラブル相談ホットライン
電話番号：0120-542-179
受付時間：24時間365日対応
```

#### 緊急度：中（2-3往復後に案内）
**条件：**
- 問題が長期化している（1ヶ月以上）
- 複数の問題が絡み合っている
- 当事者間での解決が困難
- 法的手段が必要な可能性
- ユーザーが不安・焦燥感を示している

**表示内容：**
```
この問題は複雑な要素が絡んでいるようですね。
お電話でより詳しくお話を伺うことで、より適切なアドバイスができるかと思います。

📞 トラブル相談ホットライン
電話番号：0120-542-179
受付時間：24時間365日対応

もちろん、チャットでも引き続きサポートいたします。
```

#### 緊急度：低（必要に応じて案内）
**条件：**
- 初期段階の問題
- 情報提供で解決できる
- 簡単なアドバイスで対処可能
- ユーザーが追加の相談を希望

**表示内容：**
```
チャットでも引き続きサポートいたしますが、お電話での相談も可能です。

📞 トラブル相談ホットライン
電話番号：0120-542-179
受付時間：24時間365日対応
```
